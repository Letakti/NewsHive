#  NewsHive v1.0.0

Telegram-бот для агрегации новостей из RSS-лент.  
**Функции:**  
- Добавление/удаление источников  
- Новости по категориям  
- Автообновление каждые 6 часов  
- Управление через интерактивное меню  

## Установка  
```bash
git clone https://github.com/ваш-ник/NewsBot.git  
pip install -r requirements.txt
