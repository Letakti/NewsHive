NewsHive

NewsHive — Telegram-бот на Java, который собирает новости из RSS-лент и отправляет их пользователю в удобном формате.

Зачем этот проект

Проект создан как практическая работа для изучения:

работы с Telegram Bots API;

парсинга RSS-лент;

структурирования Java‑приложения;

работы с внешними источниками данных.


Возможности

Получение новостей из RSS-лент

Отправка новостей в Telegram

Простая архитектура, удобная для расширения


Используемые технологии

Java

Telegram Bots API

RSS (XML)

Maven / Gradle (в зависимости от сборки проекта)


Как запустить

1. Клонировать репозиторий:

git clone https://github.com/Letakti/NewsHive.git


2. Создать Telegram-бота через @BotFather и получить токен


3. Указать токен бота в конфигурации проекта


4. Запустить приложение



Конфигурация

Для работы бота необходимо указать:

Telegram Bot Token

RSS-ленту (или список лент)


Рекомендуется хранить конфигурацию вне кода (например, в config.properties или .env).
