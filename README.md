# NewsHive

**NewsHive** — Telegram-бот на Python, который собирает новости из RSS-лент и отправляет их пользователю в удобном формате.

## Что умеет бот

- Получать новости по выбранному источнику.
- Получать новости по категориям.
- Отправлять случайные новости.
- Показывать подборку основных (топ) новостей.
- Управлять пользовательскими RSS-источниками (добавление/удаление/просмотр).

## Технологии

Проект реализован на Python и использует:

- `aiogram` — Telegram Bot API и обработка команд/сообщений.
- `aiohttp` — асинхронные HTTP-запросы.
- `feedparser` — парсинг RSS-лент.
- `requests` — вспомогательные HTTP-запросы.

## Запуск проекта

### 1) Установите зависимости

```bash
pip install -r requirements.txt
```

### 2) Настройте конфигурацию

Минимально нужно указать токен бота в `config.py`:

```python
TOKEN = 'ваш_токен_бота'
```

Также при необходимости настройте:

- `CHAT_ID` — ваш Telegram ID для личных тестов.
- `NEWS_SOURCES` — словарь RSS-источников.
- `NEWS_CATEGORIES` — категории и привязку к источникам.
- `GROUPS_FILE` и `USER_SOURCES_JSON` — файлы хранения данных.
- `AUTO_UPDATE_INTERVAL` — интервал автообновления.

> Если удобнее, можно хранить чувствительные параметры (например, токен) в переменных окружения и подгружать их в `config.py`.

### 3) Запустите бота

```bash
python bot.py
```

## Команды и меню

### Telegram-команды

- `/start` — запуск бота и показ главного меню.
- `/help` — список доступных команд.
- `/sources` — открыть меню управления источниками.
- `/news` — получить случайные новости.

Также в коде регистрируется команда `/top` (топ-новости), но основной сценарий получения топ-новостей сейчас реализован через кнопку меню **«📌 Основные новости»**.

### Кнопки главного меню

- **📰 Новости по источнику** — выбор источника и выдача свежих новостей.
- **📂 Новости по категории** — выбор категории и выдача новостей.
- **🎲 Рандомные новости** — случайная подборка новостей.
- **📌 Основные новости** — главные новости из ключевых источников.
- **⚙️ Управление источниками** — переход в меню управления пользовательскими RSS.

### Меню управления источниками

- **➕ Добавить источник** — добавить свой RSS-источник (название + ссылка).
- **➖ Удалить источник** — удалить ранее добавленный источник.
- **📋 Мои источники** — показать список пользовательских источников.
- **🔙 Назад** — вернуться в предыдущее меню.

## Структура (кратко)

- `bot.py` — точка входа, запуск polling и регистрация команд.
- `handlers.py` — обработчики команд и кнопок.
- `parser.py` — работа с RSS и формирование новостных выдач.
- `keyboards.py` — клавиатуры и кнопки меню.
- `config.py` — настройки бота и источников.
